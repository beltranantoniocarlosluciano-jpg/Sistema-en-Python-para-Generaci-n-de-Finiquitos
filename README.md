# Sistema-en-Python-para-Generaci-n-de-Finiquitos
Script en Python para automatizar la generación de finiquitos laborales. Incluye un menú interactivo en consola y carga datos desde Excel para gestionar información del personal. Proyecto en desarrollo enfocado en mejorar procesos administrativos, con planes de integrar POO, base de datos y exportación automática de documentos.
